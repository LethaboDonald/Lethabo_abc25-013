# BSBS Online Voting System

A secure, user-friendly digital platform for electing the Student Representative Council (SRC) at the Botswana School of Business Sciences.

## Features
- Secure student login
- Candidate profiles with manifesto
- Digital ballot (one vote per position)
- Real-time counting & results dashboard
- Admin panel for managing voters & candidates

## Tech Stack
- Frontend: HTML5, CSS3, JavaScript
- Backend: ASP.NET Core MVC
- Database: SQL Server

## Setup
1. Clone the repo
2. Open `index.html` in your browser for frontend preview
3. Backend setup instructions coming soon
